投资笔记

投资是一种市场行为。
投资的组成要素有投资标的，买入和卖出的交易行为。
投资标的区别于供消费的商品。二者最大的区别在于流动性，也就是是否有大量潜在买家。举例来说，中国的普通住房在2019年之前是投资标的，2023年之后是大宗消费品。
交易行为的目的是逐利。以股票为例，低买高卖追求价格差距是最常见的交易行为。
投资的核心在于标的选择和价格判断。
市场上可选的投资标的罗列如下：
股票，债券，基金，期货，实物贵金属，广义收藏品，商业地产，
价格判断是投资中最困难，最迷人的部分。
先考虑整体价格水平。对于整个投资市场而言，价格水平遵循最基本的供需原理。作为供应端，投资标的的变化往往是缓慢的，而需求端的变化相比之下要快的多。市场的总体价格水平基本上可以看做由需求端决定。
影响投资需求的第一个因素是流动性，即市场上可供投资的资金总量。过去半个世纪，主要资产价格都涨了很多倍，原因在于货币供应量大量增加，比财富增加的更快。以美股为例，标普指数在过去半个世纪涨了倍，而美元流通量涨了倍。
观察市场流动性，需要观察如下指标：美联储利率，美国银行隔夜拆借利率，美国财政部存款，中国央行利率，中国银行隔夜拆借利率。
影响投资需求的第二个因素是预期，即投资者结合客观因素，主观因素，对价格水平的判断。当市场中的所有参与者都认为某个资产价值低估，那么该资产的价格必然上涨。典型案例如比亚迪，电车业务爆炸式增长，带来营收大幅上升，此时公司股价飙升是可预期的。
影响预期的客观因素，需要观察以下指标：资产价格对比历史价格所处的价格水平，资产的盈利能力，资产交易参与者数量，
影响预期的主观因素，需要观察以下指标：资产的抗风险属性与市场对风险的预期，资产的未来盈利故事与市场的认可度，资产交易者的群聚效应与踩踏效应
