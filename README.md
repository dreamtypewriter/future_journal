投资笔记

投资是一种市场行为。
投资的组成要素有投资标的，买入和卖出的交易行为。
投资标的区别于供消费的商品。二者最大的区别在于流动性，也就是是否有大量潜在买家。举例来说，中国的普通住房在2019年之前是投资标的，2023年之后是大宗消费品。
交易行为的目的是逐利。以股票为例，低买高卖追求价格差距是最常见的交易行为。
投资的核心在于标的选择和价格判断。
市场上可选的投资标的罗列如下：
股票，债券，基金，期货，实物贵金属，广义收藏品，商业地产，
价格判断是投资中最困难，最迷人的部分。
先考虑整体价格水平。对于整个投资市场而言，价格水平遵循最基本的供需原理。作为供应端的了，投资标的变化往往是缓慢的，而需求端的变化相比之下要快的多。市场的总体价格水平基本上可以看做由需求端决定。
影响投资需求的第一个因素是流动性，即市场上可供投资的资金总量。
